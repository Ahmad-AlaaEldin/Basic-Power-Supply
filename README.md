# Basic Regulated DC Power Supply

## 📌 Project Overview
This repository contains the design, simulation, and testing details of a basic regulated DC power supply. The main objective of this project was to design a reliable circuit capable of delivering a stable output voltage to power various electronic circuits safely.

## ⚙️ Key Features & Achievements
* **Circuit Design:** Successfully designed a regulated DC supply that delivers a consistent and stable output voltage.
* **Performance Testing & Debugging:** Rigorously tested the output stability and debugged the circuit's performance under varying load conditions to ensure reliability.

## 🛠️ Tools & Components Used
* **Hardware Components:**  Transformer, Diodes/Bridge Rectifier, Capacitors, Voltage Regulator IC etc....
* **Testing Equipment:**  Oscilloscope, Digital Multimeter
